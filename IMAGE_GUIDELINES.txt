REQUIRED IMAGES FOR YOUR WEBSITE
=================================

You'll need to add these images to your repository:

1. PROFILE PHOTO
   Filename: your-photo.jpg
   Recommended size: 800x800px (square)
   Format: JPG or PNG
   Description: Professional headshot with good lighting
   Location: Root directory

2. RESEARCH IMAGES
   Filenames: 
   - research-1.jpg (Featured research)
   - research-2.jpg (Second project)
   - research-3.jpg (Third project)
   
   Recommended size: 1200x800px (landscape)
   Format: JPG or PNG
   Description: Visualizations, graphs, satellite imagery, or field photos
   Location: Root directory or /images/ folder

IMAGE TIPS:
-----------
✓ Use high-quality images (but compress them!)
✓ Keep file sizes under 500KB each
✓ Use consistent aspect ratios
✓ Ensure good contrast and visibility
✓ Use descriptive alt text

COMPRESSION TOOLS:
------------------
- TinyPNG: https://tinypng.com
- Squoosh: https://squoosh.app
- ImageOptim: https://imageoptim.com (Mac only)

FREE IMAGE SOURCES:
-------------------
- Unsplash: https://unsplash.com (Free high-quality photos)
- Pexels: https://pexels.com (Free stock photos)
- NASA Image Gallery: https://images.nasa.gov (Satellite/space imagery)
- USGS: https://www.usgs.gov/media (Earth science imagery)

YOUR OWN RESEARCH IMAGES:
-------------------------
Best options:
- Satellite imagery visualizations
- Graphs and charts from your papers
- Field work photographs
- GIS maps and spatial analysis
- Hydrological models visualization
- Remote sensing data products

PLACEHOLDER IMAGES:
-------------------
Until you have your images ready, you can use:
- https://via.placeholder.com/800x800 (profile photo)
- https://via.placeholder.com/1200x800 (research images)

Just add these URLs temporarily in the HTML until you replace with real images.

NEXT STEPS:
-----------
1. Gather or create your images
2. Compress them using tools above
3. Rename them as specified
4. Upload to your GitHub repository
5. Update paths in index.html if needed
